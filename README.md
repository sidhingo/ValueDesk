# Value Desk
### PE Value Creation Intelligence Platform

A professional analytics tool for private equity practitioners. Built to model how operational levers translate into fund returns.

**Live tool:** [value-desk.vercel.app](https://value-desk.vercel.app)

---

## Modules

**Fund Benchmarking** — Input TVPI, DPI, and IRR against vintage year and strategy to identify quartile ranking vs institutional peer groups.

**Deal Returns** — Model MOIC and IRR from entry to exit, factoring in transaction fees and interim distributions.

**Value Creation** — Simulate how revenue growth, margin expansion, and multiple re-rating combine to drive enterprise value over a hold period.
