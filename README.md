# Value Desk
### PE Value Creation Intelligence Platform

A professional analytics tool built for private equity practitioners. Value Desk models how operational levers, fund performance, and deal structures translate into returns — with live fund data from an institutional data provider.

**Live tool:** [value-desk.vercel.app](https://value-desk.vercel.app)

---

## Modules

**Fund Benchmarking** — Search 21,000+ institutional funds and benchmark TVPI, DPI, and IRR against vintage year peer groups. Outputs quartile rankings, realization mix, and LP strategy recommendations. Powered by live fund data.

**Deal Returns** — Model MOIC and IRR from entry to exit, factoring in transaction fees and interim distributions. Built for deal-level return analysis.

**Value Creation** — Simulate how revenue growth, margin expansion, and multiple re-rating combine to drive enterprise value over a hold period. Outputs total value created and performance upside.

**100-Day Plan** — Generate a structured operational framework for the first 100 days post-close, assembled dynamically from value creation thesis, entry situation, sector, and company size. Covers 448+ distinct plan combinations.
